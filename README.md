# ValueRank

ValueRank evaluates how AI models prioritize moral values when faced with tradeoff prompts.

## Where things live

- **`cloud/`** — the active product (API, web, workers, shared packages)
- **`docs/`** — architecture, PRD, canonical glossary, workflow rules, feature-run records
- **`specs/`** — feature specifications
- **`scripts/`** — maintenance and operational scripts
- **`AGENTS.md`** — shared working contract for all AI agents in this repo
- **`STATUS.md`** — current goals, active bugs, and recently completed work
