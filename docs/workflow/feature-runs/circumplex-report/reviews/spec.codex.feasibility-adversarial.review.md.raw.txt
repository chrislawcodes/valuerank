## Residual Risks
- The MDS requirement still needs an explicit anchoring rule. Without one, the empirical embedding can rotate or flip arbitrarily, which makes the theoretical circle and off-axis tooltips hard to interpret.
- The spec also needs a final decision on whether the backend should be cached/materialized or recomputed on demand, because the current code path is snapshot-oriented and the new report’s aggregation scope is much wider.
